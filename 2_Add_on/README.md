# Add-on analyses

This directory contains revision-stage analyses built on top of the core master tables generated by `1_python_script_main/`.

## Analyses included

- structure preservation (Mantel, Procrustes, neighborhood overlap)
- LD redundancy and information-capacity summaries
- trait-versus-ancestry confounding checks
- PR/TARS overlap validation
- benchmarking against alternative marker-selection strategies

## Expected inputs

By default, the script reads from `../1_python_script_main/master/` when launched from the repository root with `--project-root .`.

Required files:

- `icgt_geno_master.csv`
- `icgt_pheno_master.csv`
- `barcode_panel_32_stats.csv`

Optional files:

- `pr_traits_master.csv`
- `accession_overlap_auto.csv`

## Usage

Using pip:

```bash
pip install -r 2_Add_on/requirements.txt
python 2_Add_on/run_analyses.py --project-root .
```

Using conda:

```bash
cd 2_Add_on
conda env create -f environment.yml
conda activate cacaocipher-addon
python run_analyses.py --project-root ..
```

## Outputs

Outputs are written to `2_Add_on/addon_outputs/` by default. Tables are saved as CSV. Figures are saved as PDF and PNG.

## Notes

- Figure and table numbering are not hard-coded.
- Random seeds are fixed by default for reproducibility.
